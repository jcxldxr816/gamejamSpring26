# On Record
My Truman ACM Gamejam submission for Spring '26!

**On Record** is *a remix* of Jet Set Radio, a game about moving quickly and graffiti.

I was inspired by Jet Set Radio, 1984, and The Matrix. This was an idea that I'd had for a few months, and I'm glad I finally had an excuse to start on it! I plan on continuing to work on the game in the future. There are a lot of features I had planned that I didn't have time to add.

## Future Plans
I want to add more fun movement options, like wall climbing, wall running, sliding, and surfing (Counter Strike).
I would also like to add enemies, and citizen NPCs.
Of course, the game would be much better if it had actual assets and art supporting it as well.
